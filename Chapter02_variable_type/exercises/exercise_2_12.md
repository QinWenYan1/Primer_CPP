# 📚 C++ Primer 2.2 课后练习题解答

## 🔍 Exercise 2.12: 标识符有效性

**(a) `int double = 3.14;`** → **无效** (关键字)

**(b) `int _;`** → **有效** (但不推荐)

**(c) `int catch-22;`** → **无效** (包含连字符)
- 正确：`int catch_22;`

**(d) `int 1_or_2 = 1;`** → **无效** (数字开头)  
- 正确：`int one_or_two = 1;`

**(e) `double Double = 3.14;`** → **有效** (大小写敏感)
