### 🔹 Exercise 4.1

表达式：

```cpp
5 + 10 * 20 / 2
```

1. 运算符优先级：

   * `*` 和 `/` 的优先级高于 `+`。
   * `*` 和 `/` 同级，按 **左结合**。

2. 计算顺序：

   * `10 * 20 = 200`
   * `200 / 2 = 100`
   * `5 + 100 = 105`

✅ **答案：105**

---

### 🔹 Exercise 4.2

**(a)**

```cpp
* vec.begin()
```

* `.`（点运算符）优先级高于 `*`（解引用）。
* 所以先计算 `vec.begin()`，再解引用结果。

👉 加括号形式：

```cpp
*(vec.begin())
```

---

**(b)**

```cpp
* vec.begin() + 1
```

* 同理，先计算 `vec.begin()`。
* 然后解引用 `*`。
* 最后执行 `+ 1`。

👉 加括号形式：

```cpp
(*(vec.begin())) + 1
```

---

## ✅ 最终答案

* **4.1** → `105`
* **4.2 (a)** → `*(vec.begin())`
* **4.2 (b)** → `(*(vec.begin())) + 1`
