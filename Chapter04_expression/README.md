## 💡 核心速查

**运算符优先级（从高到低）**
```cpp
::                          // 作用域解析
. -> [] () ++ --           // 成员访问、下标、函数调用
++ -- + - ! ~ * & sizeof   // 一元运算符
.* ->*                     // 成员指针访问
* / %                       // 乘除模
+ -                         // 加减
<< >>                      // 移位
< <= > >=                  // 关系
== !=                       // 相等性
&                           // 按位与
^                           // 按位异或
|                           // 按位或
&&                          // 逻辑与
||                          // 逻辑或
?:                          // 条件
= += -= *= /= %= &= |=     // 赋值
^= <<= >>=                 // 复合赋值
,                           // 逗号
```

**常用运算符示例**
```cpp
// 算术运算
int a = 10 + 5;     // 15
int b = 10 / 3;     // 3 (整数除法)
int c = 10 % 3;     // 1 (取模)

// 关系与逻辑
bool d = (a > b) && (c < 5);  // 短路求值
bool e = !(a == b);             // 逻辑非

// 位运算
int f = 0b1010 & 0b1100;  // 0b1000 (8)
int g = 0b1010 | 0b1100;   // 0b1110 (14)
int h = 1 << 3;            // 8 (左移3位)

// 条件运算符
int max = (a > b) ? a : b;  // 取较大值

// sizeof
size_t sz = sizeof(int);     // 4 (通常)
size_t arr_sz = sizeof(arr) / sizeof(arr[0]);  // 数组元素个数
```

**类型转换**
```cpp
// C++ 风格显式转换
int i = static_cast<int>(3.14);     // 3
double d = static_cast<double>(5); // 5.0

// const_cast - 去除 const
const char* p = "hello";
char* q = const_cast<char*>(p);

// 避免 C 风格转换
int bad = (int)3.14;  // 不推荐，不明确
```
