### Exercise 12.25 动态数组的删除

**原题：** Given the following new expression, how would you delete pa?
`int *pa = new int[10];`

**你的答案：** 如果我们要删除pa如下:`delete [] pa`

**批改：** ⭐⭐⭐⭐⭐ 5/5 完美答案！你完全理解了动态数组内存释放的正确方法。

**详细分析：**
- ✅ **概念理解正确**：你准确理解了`new[]`分配的数组必须使用`delete[]`释放
- ✅ **语法使用准确**：`delete [] pa`是完全正确的语法格式
- ✅ **代码规范性良好**：使用了正确的括号位置和语法结构
- ⚠️ **注意语句完整性**：在实际代码中，删除语句应该以分号结束：`delete [] pa;`

**正确答案：**
```cpp
delete [] pa;
```
